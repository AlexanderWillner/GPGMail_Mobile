RSA Public Key Encryption Test in JavaScript

2005 Herbert Hanewinkel
[Description][Test]

This page contains my Javascript version of RSA Public Key encryption.
It is a clone of John Hanna's RSA implementation with a few minor modifications to adapt it for use with GnuPG's multi precision integers.

The implementation consists of the following Javascript sources:

  • rsa.js -- RSA encryption/decryption.
  • keygen.js -- key generation routines for RSA test page.
